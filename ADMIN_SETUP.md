# راه‌اندازی پنل مدیریت PesteOnline — نسخه اول

## امکانات نسخه اول
- ورود اختصاصی مدیر با Supabase Auth
- بازیابی رمز عبور با ایمیل
- داشبورد راست‌چین و واکنش‌گرا
- ویرایش سریع تمام قیمت‌ها
- ثبت منبع استعلام برای هر محصول
- پیش‌نمایش قبل از انتشار
- تأیید قیمت‌های بدون تغییر برای روز جدید
- ثبت روزانه تاریخچه قیمت‌ها
- ساختار اولیه مقالات، تحلیل بازار، طراحی سایت و آمار

## مرحله ۱: ساخت جدول‌های دیتابیس
در Supabase وارد `SQL Editor` شو و فایل زیر را اجرا کن:

`supabase/admin_v1.sql`

## مرحله ۲: ساخت حساب مدیر
در Supabase وارد این مسیر شو:

`Authentication → Users → Add user`

ایمیل خودت و یک رمز قوی ثبت کن. ثبت‌نام عمومی در پنل وجود ندارد.

## مرحله ۳: متغیرهای محیطی
فایل `.env` باید شامل این دو مقدار باشد:

```env
VITE_PUBLIC_SUPABASE_URL=...
VITE_PUBLIC_SUPABASE_ANON_KEY=...
```

## مرحله ۴: اجرا
```bash
npm install
npm run dev
```

پنل در این آدرس باز می‌شود:

`http://localhost:3000/admin`

## انتشار روی GitHub و Cloudflare
```bash
git add .
git commit -m "Add PesteOnline admin panel v1"
git push origin main
```

Cloudflare Pages پس از Push، نسخه جدید را خودکار Deploy می‌کند.

## نکته داده آزمایشی
اگر جدول `pistachio_prices` خالی باشد یا اتصال Supabase آماده نباشد، صفحه قیمت‌ها برای نمایش رابط از داده آزمایشی استفاده می‌کند. بعد از اجرای SQL و وجود داده واقعی، اطلاعات مستقیماً از Supabase خوانده می‌شوند.
